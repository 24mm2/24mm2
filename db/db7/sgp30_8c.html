◆ sgp_get_iaq_baseline()

s16 sgp_get_iaq_baseline ( u32 *  baseline)

sgp_get_iaq_baseline() - read out the baseline from the chip

The IAQ baseline should be retrieved and persisted for a faster sensor startup. See sgp_set_iaq_baseline() for further documentation.

A valid baseline value is only returned approx. 60min after a call to sgp_iaq_init() when it is not followed by a call to sgp_set_iaq_baseline() with a valid baseline. This functions returns STATUS_FAIL if the baseline value is not valid.

@baseline: Pointer to raw u32 where to store the baseline If the method returns STATUS_FAIL, the baseline value must be discarded and must not be passed to sgp_set_iaq_baseline().

Return: STATUS_OK on success, else STATUS_FAIL
